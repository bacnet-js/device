    Interface BDApplicationTagValueType

    Map between BACnet Application Tags and native JavaScript types.

    This interface defines the mapping between each BACnet ApplicationTag and its corresponding JavaScript/TypeScript type. This mapping is used throughout the library to ensure type safety when working with BACnet values.

    Entries mapping to the never type are not yet supported by this implementation.
