    Enumeration ApplicationTag

    BACnet application tag values

    This enumeration represents the standard application tags in BACnet as defined in the BACnet standard. Application tags identify the data type of a value in BACnet communications.

    Application tags are used in BACnet property values to indicate the data type of the value being communicated.
