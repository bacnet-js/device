    Enumeration Reliability

    BACnet reliability values

    This enumeration represents the possible reliability states of a BACnet object as defined in the BACnet standard. It is used in the RELIABILITY property of BACnet objects to indicate whether the Present_Value property is reliable and, if not, why it is unreliable.
