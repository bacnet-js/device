    Enumeration ObjectType

    BACnet object type values

    This enumeration represents the standard object types in BACnet as defined in the BACnet standard. Each value identifies a specific type of object that can exist in a BACnet system.

    The object type is a fundamental component of an object identifier, which uniquely identifies objects in a BACnet network.
