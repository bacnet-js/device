    Enumeration ErrorCode

    BACnet error code values

    This enumeration represents the standard error codes in BACnet as defined in the BACnet standard. Error codes provide specific information about what kind of error has occurred within a given error class.

    Error codes are used in conjunction with error classes to provide detailed information about the nature of an error that has occurred.
