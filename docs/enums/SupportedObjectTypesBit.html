    Enumeration SupportedObjectTypesBit

    Enumeration of the standard BACnet object type bit positions

    These values represent the bit positions in a Protocol_Object_Types_Supported bitstring as defined in the BACnet standard. Each bit corresponds to a specific object type that a BACnet device may support.
